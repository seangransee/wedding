# Wedding Website

Next.js 15 app for Sean and Lexi's wedding website.

## Structure
- `src/app/layout.tsx` - Root layout with fonts (Geist, Dancing Script)
- `src/app/page.tsx` - Home page with wedding title
- Uses Tailwind CSS and TypeScript

## Commands
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
